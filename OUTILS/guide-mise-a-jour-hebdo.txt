🔄 GUIDE : Mise à jour hebdomadaire de la base de données

📅 CHAQUE JEUDI après le Zoom FAQ :

1️⃣ Regarder/revoir le Zoom et noter :
   • Nouvelle pathologie abordée
   • Produits recommandés
   • Posologies précises
   • Précautions mentionnées
   • Conseils complémentaires

2️⃣ Ajouter dans `pathologies.json` :
   - Copier le template ci-dessous
   - Remplacer les [...] par les infos réelles
   - Ajouter en fin de tableau "pathologies"
   - Incrémenter l'ID (ex: "nouvelle-pathologie")

3️⃣ Mettre à jour `base-donnees-pathologies.html` :
   - Ajouter la nouvelle fiche dans le tableau JavaScript `pathologies`
   - Mettre à jour la date dans le footer

4️⃣ Redéployer sur Netlify
