💡 À quoi sert cet outil ?

Cet outil génère automatiquement un fichier index.html personnalisé pour chaque membre qui veut devenir autonome. Il suffit de remplir les informations du membre et le générateur crée le code complet avec SES coordonnées !

📋 Processus en 4 étapes :

  1. Remplis le formulaire avec les infos du membre
  2. Clique sur "🔧 Générer le tunnel"
  3. Télécharge le fichier index.html généré
  4. Envoie le fichier au membre + guide de déploiement

📦 Instructions pour le membre :

  1. Télécharge le fichier index.html généré
  2. Crée un dossier sur ton ordinateur (ex: mon-tunnel/)
  3. Place le fichier index.html dans ce dossier
  4. Va sur Netlify → app.netlify.com
  5. Glisse le dossier dans la zone de déploiement
  6. Attends 30 secondes → Ton site est en ligne !
  7. Personnalise le nom dans Site Settings → Domain Management
  8. Teste ton tunnel avec ?parrain=tonnom
