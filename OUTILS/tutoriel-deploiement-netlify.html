🚀 Tutoriel Déploiement Netlify

Guide pas à pas pour mettre ton site en ligne

🖨️ Imprimer / PDF ← Retour au Support
⏱️ Temps estimé : 10-15 minutes
💰 Coût : 100% gratuit
📋 Pré-requis : Un compte Netlify + ton fichier index.html

📁 Étape 1 : Préparer ton dossier

  1. Crée un nouveau dossier sur ton Bureau (ex: "mon-site-atomy")
  2. Dans ce dossier, mets ton fichier index.html
  3. Vérifie que le fichier s'appelle bien index.html (en minuscules)
  4. ⚠️ Important : Le fichier doit être à la RACINE du dossier, pas dans un sous-dossier !
📁 mon-site-atomy/
└── 📄 index.html

🌐 Étape 2 : Se connecter à Netlify

  1. Va sur app.netlify.com
  2. Connecte-toi avec ton compte (ou crée-en un si ce n'est pas fait)
  3. Une fois connecté, tu arrives sur ton tableau de bord

🚀 Étape 3 : Déployer ton site

  1. Clique sur "Add new site" → "Deploy manually"
  2. Une zone apparaît : "Drag and drop your site folder here"
  3. Glisse ton dossier COMPLET (pas les fichiers un par un !) dans cette zone
  4. Attends 30-60 secondes pendant le déploiement
  5. ✅ C'est en ligne ! Netlify te donne un lien du type : https://xxxxxx.netlify.app
🖼️ [Zone de drag & drop Netlify]

🔗 Étape 4 : Personnaliser ton nom de site

  1. Dans Netlify, clique sur ton site
  2. Va dans "Site settings" (menu gauche, icône ⚙️)
  3. Clique sur "Domain management"
  4. Descends à "Options" → "Edit site name"
  5. Change le nom (ex: laurent-atomy)
  6. Clique sur "Save"
  7. ✅ Ton nouveau lien : https://laurent-atomy.netlify.app
⚠️ Attention :
  • Le nom doit être UNIQUE sur tout Netlify
  • Uniquement lettres minuscules, chiffres, tirets (pas d'espaces)
  • L'ancien lien ne fonctionnera plus après le changement

✅ Étape 5 : Tester ton site

  1. Ouvre ton nouveau lien dans un navigateur
  2. Teste avec ?parrain=tonnom à la fin de l'URL
  3. Vérifie que le formulaire fonctionne
  4. Remplis le formulaire avec ton email de test
  5. Vérifie que tu reçois l'email de confirmation
  6. ✅ Tout fonctionne ? C'est prêt à partager !

🔄 Mettre à jour ton site

  1. Modifie ton fichier index.html sur ton PC
  2. Enregistre les modifications
  3. Va sur Netlify → Ton site → Onglet "Deploys"
  4. En haut : "Need to update your site? Drag and drop your site folder here"
  5. Glisse ton dossier COMPLET à nouveau
  6. Attends 30 secondes → ✅ Site mis à jour !
💡 Astuces Pro :
  • Garde toujours une copie de ton fichier index.html sur ton PC
  • Fais des tests après chaque modification
  • Utilise un éditeur de code comme VS Code pour modifier ton HTML
  • Netlify met environ 30 secondes pour déployer — sois patient !
⚠️ Problèmes fréquents :
  • "No index.html found" → Ton fichier n'est pas à la racine du dossier
  • "Page not found" → Vérifie que le fichier s'appelle bien index.html (minuscules)
  • Le formulaire ne marche pas → Vérifie ta clé API Brevo dans le code
  • L'email ne part pas → Vérifie ton automatisation dans Brevo
