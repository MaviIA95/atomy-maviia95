🎨 Guide : Modifier les Couleurs

Personnalise facilement les couleurs de ton site ATOMY.

📍 Où modifier les couleurs ?

Ouvre ton fichier index.html ou support.html dans un éditeur de texte (Bloc-notes, NotePade++, VS Code...).

Cherche cette section tout en haut du CSS :

:root {
    --primary-color: #2c5282;        /* Bleu principal */
    --primary-light: #4a5568;        /* Bleu secondaire */
    --accent-color: #3182ce;         /* Bleu accent */
    --background: #f7fafc;           /* Fond de page */
    --surface: #ffffff;              /* Surfaces (cartes) */
    --text-primary: #1a202c;         /* Texte principal */
    --text-secondary: #4a5568;       /* Texte secondaire */
    --font-base: 18px;               /* Taille de base */
}

🎨 Palette de couleurs suggérée (Accessibilité)

  • #2c5282 - Bleu doux (primaire)
  • #4a5568 - Bleu gris (secondaire)
  • #f7fafc - Fond doux (moins éblouissant)
  • #1a202c - Texte foncé (meilleur contraste)

📏 Ajuster la taille du texte

Modifie --font-base: 18px; :

  • 16px = taille normale
  • 18px = confortable (recommandé)
  • 20px = très lisible

💾 Sauvegarder et tester

  1. Modifie les valeurs dans :root
  2. Enregistre le fichier
  3. Ouvre-le dans ton navigateur pour voir le résultat
  4. Si ça te plaît, redéploie sur Netlify
