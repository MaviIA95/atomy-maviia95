📧 Guide Complet Brevo

De la création du compte à l'automatisation des emails

← Retour au Support
⏱️ Temps estimé : 25-35 minutes
💰 Coût : Gratuit jusqu'à 300 emails/jour
📋 Pré-requis : Une adresse email valide + ton site Netlify prêt

📝 Étape 1 : Créer ton compte Brevo

  1. Ouvre ton navigateur et va sur : www.brevo.com
  2. Clique sur le bouton "Créer un compte" (en haut à droite)
  3. Choisis "Je veux envoyer des emails"
  4. Remplis le formulaire :
    • Prénom : Ton prénom
    • Nom : Ton nom
    • Email professionnel : Ton email (ex: karine@email.com)
    • Mot de passe : Un mot de passe fort (12+ caractères)
    • Téléphone : Ton numéro avec indicatif (ex: +33612345678)
    • Pays : Ton pays de résidence
  5. Coche la case pour accepter les conditions
  6. Clique sur "Créer mon compte"

✅ Étape 2 : Vérifier ton email

  1. Ouvre ta boîte email
  2. Trouve l'email de Brevo (objet : "Vérifiez votre adresse email")
  3. Clique sur le bouton "Vérifier mon email" dans l'email
  4. ✅ Ton compte est maintenant activé !

🔑 Étape 3 : Trouver ta clé API (ESSENTIEL)

⚠️ Cette étape est cruciale : Ta clé API permet à ton site de communiquer avec Brevo.

  1. Connecte-toi sur app.brevo.com
  2. Dans le menu gauche, clique sur "Paramètres" (icône roue crantée ⚙️)
  3. Dans le sous-menu, clique sur "Clés API"
  4. Clique sur "Générer une nouvelle clé API"
  5. Donne un nom à ta clé (ex: Tunnel ATOMY - Karine)
  6. Clique sur "Générer"
  7. ✅ Ta clé apparaît : elle commence par xkeysib-
  8. COPIE-LA IMMÉDIATEMENT et colle-la dans un endroit sûr (Bloc-notes, gestionnaire de mots de passe...)
⚠️ Attention :
• Cette clé est comme un mot de passe : ne la partage jamais !
• Une fois générée, tu ne pourras plus la revoir dans Brevo (seulement la recréer)
• Si tu la perds, tu devras en générer une nouvelle et mettre à jour ton code
xkeysib-a7972d18d1eefe6a142f3c410aa......................................-XXXXXX

📋 Étape 4 : Créer ta liste de contacts

  1. Dans Brevo, menu gauche → "Contacts" → "Listes"
  2. Clique sur "Créer une liste"
  3. Remplis :
    • Nom de la liste : Leads ATOMY - Karine
    • Description : Prospects du tunnel ATOMY de Karine
  4. Clique sur "Créer"
  5. ✅ Note l'ID de la liste (un nombre, ex: 2) → Tu en auras besoin dans ton code

🏷️ Étape 5 : Créer tes attributs personnalisés

⚠️ Important : Ces attributs doivent être créés AVANT d'envoyer des données depuis ton site.

  1. Dans Brevo, menu gauche → "Contacts" → "Paramètres" (roue crantée ⚙️)
  2. Clique sur "Attributs des contacts"
  3. Pour CHAQUE attribut ci-dessous, clique sur "Créer un attribut" et remplis :
    • Nom interne : PRENOM (en MAJUSCULES, sans espaces)
    • Type : Texte
    • Description : Prénom du contact
    • Clique sur "Créer"

📋 Liste des attributs à créer :

PRENOM (Texte)
NOM (Texte)
TELEPHONE (Texte)
PAYS (Texte)
OBJECTIF (Texte)
PARRAIN (Texte) ← Pour tracker qui amène les leads
SOURCE (Texte) ← Pour savoir d'où vient le lead
DATE_INSCRIPTION (Texte) ← Optionnel, Brevo le fait auto
💡 Astuce :
Les noms d'attributs doivent correspondre EXACTEMENT à ceux dans ton code JavaScript (majuscules, pas d'espaces).

📧 Étape 6 : Créer ton template d'email

  1. Dans Brevo, menu gauche → "Campagnes" → "Templates"
  2. Clique sur "Créer un template" → "Template d'email"
  3. Remplis les infos de base :
    • Email de l'expéditeur : Ton email (ex: karine@email.com)
    • Nom de l'expéditeur : Karine - ATOMY
    • Objet : ✅ Votre réservation ATOMY est confirmée !
    • Aperçu du texte : Votre place au webinaire est réservée...
  4. Clique sur "Ajouter du contenu"
  5. Choisis "Éditeur HTML personnalisé" (ou "Code HTML")
  6. Colle le code HTML de ton email de confirmation (fourni par Laurent)
  7. Clique sur "Enregistrer et quitter"
  8. ✅ Note l'ID du template (un nombre, ex: 3) → Tu en auras besoin dans ton code

🔄 Étape 7 : Créer ton automatisation

  1. Dans Brevo, menu gauche → "Automatisations"
  2. Clique sur "Créer un workflow"
  3. Choisis : "Déclencheur : Contact ajouté à une liste"
  4. Sélectionne ta liste : Leads ATOMY - Karine
  5. Clique sur "Suivant"
  6. Clique sur "Ajouter une action"
  7. Choisis : "Envoyer un email"
  8. Sélectionne ton template : Confirmation Webinaire ATOMY
  9. Clique sur "Enregistrer"
  10. ✅ Active le workflow (bouton en haut à droite → "Actif")
💡 Test rapide :
Ajoute manuellement un contact de test à ta liste Brevo → Vérifie que l'email automatique part bien.

🔗 Étape 8 : Mettre à jour ton code index.html

⚠️ Cette étape relie ton site à Brevo :

  1. Ouvre ton fichier index.html dans un éditeur de texte
  2. Cherche la fonction handleSubmit() (vers la ligne ~900)
  3. Remplace la clé API :
    'api-key': 'xkeysib-a7972d18d1eefe6a142f3c41xxxx...............' ← Remplace par TA clé
  4. Vérifie que listIds: [2] correspond à l'ID de TA liste Brevo
  5. Vérifie que templateId: 3 correspond à l'ID de TON template
  6. Enregistre le fichier

✅ Étape 9 : Tester l'intégration complète

  1. Déploie ton site mis à jour sur Netlify
  2. Ouvre ton site : https://karine-atomy.netlify.app/?parrain=karine
  3. Remplis le formulaire avec un email de test
  4. Soumets le formulaire
  5. Vérifie dans Brevo :
    • ✅ Le contact apparaît dans ta liste "Leads ATOMY - Karine"
    • ✅ L'attribut PARRAIN est bien rempli avec "Karine"
    • ✅ Les autres attributs (PRENOM, EMAIL, etc.) sont corrects
  6. Vérifie dans ta boîte email :
    • ✅ Tu reçois l'email de confirmation automatique
    • ✅ Le template est bien celui que tu as créé
    • ✅ Les variables ({{params.PRENOM}}) sont bien remplacées
  7. ✅ Tout fonctionne ? Ton système est prêt !
⚠️ Problèmes fréquents et solutions :
  • "Erreur 401 / Unauthorized" → Ta clé API est incorrecte ou expirée → Recrée-la dans Paramètres → Clés API
  • "Erreur 400 / Bad Request" → Un attribut n'existe pas dans Brevo → Crée-le dans Paramètres → Attributs
  • L'email ne part pas → Vérifie que l'automatisation est "Active" et que le template existe
  • Le contact n'apparaît pas → Attends 1-2 minutes (Brevo peut être lent) ou rafraîchis la page
  • Les variables {{params.PRENOM}} ne sont pas remplacées → Vérifie que les noms dans params: {} correspondent aux attributs Brevo
