🌐 Guide Complet Netlify

De la création du compte au déploiement de ton site

← Retour au Support
⏱️ Temps estimé : 15-20 minutes
💰 Coût : 100% gratuit (plan Starter)
📋 Pré-requis : Une adresse email valide

📝 Étape 1 : Créer ton compte Netlify

  1. Ouvre ton navigateur et va sur : app.netlify.com
  2. Clique sur le bouton "Sign up" (en haut à droite)
  3. Choisis une méthode d'inscription :
    • Email : Entre ton email et crée un mot de passe
    • GitHub : Si tu as un compte GitHub (recommandé)
    • GitLab ou Bitbucket : Autres options
  4. Remplis le formulaire :
    • Full name : Ton nom complet (ex: Karine Martin)
    • Email : Ton email professionnel
    • Password : Un mot de passe fort (12+ caractères)
  5. Coche la case pour accepter les conditions
  6. Clique sur "Create account"

✅ Étape 2 : Vérifier ton email

  1. Ouvre ta boîte email
  2. Trouve l'email de Netlify (objet : "Verify your email address")
  3. Clique sur le bouton "Verify email" dans l'email
  4. ✅ Ton compte est maintenant activé !
📧 [Email de vérification Netlify]
Clique sur "Verify email" pour activer ton compte

🏠 Étape 3 : Découvrir ton tableau de bord

  1. Après vérification, tu arrives sur ton Dashboard Netlify
  2. Tu verras un message : "You don't have any sites yet"
  3. Clique sur "Add new site" → "Deploy manually"
  4. Une zone apparaît : "Drag and drop your site folder here"
🖼️ [Zone de drag & drop Netlify]
C'est ici que tu glisseras ton dossier pour déployer

📁 Étape 4 : Préparer ton dossier de site

  1. Sur ton ordinateur, crée un nouveau dossier (ex: karine-atomy)
  2. Dans ce dossier, mets ton fichier index.html
    • ⚠️ Le fichier DOIT s'appeler index.html (en minuscules)
    • ⚠️ Il doit être à la RACINE du dossier (pas dans un sous-dossier)
  3. (Optionnel) Ajoute d'autres fichiers si besoin (CSS, images, etc.)
📁 karine-atomy/
└── 📄 index.html ← OBLIGATOIRE, à la racine

🚀 Étape 5 : Déployer ton site (première fois)

  1. Dans Netlify, assure-toi d'être sur la page "Add new site"
  2. Glisse ton dossier COMPLET dans la zone "Drag and drop your site folder here"
    • ⚠️ Glisse le DOSSIER, pas les fichiers un par un !
    • ⚠️ Attends que la barre de progression atteigne 100%
  3. Netlify upload et déploie automatiquement (30-60 secondes)
  4. ✅ Quand c'est fini, tu vois : "Published" avec un lien du type :
    https://random-name-12345.netlify.app
  5. Clique sur le lien pour voir ton site en ligne !

🔗 Étape 6 : Personnaliser le nom de ton site

  1. Dans Netlify, clique sur ton site (dans la liste "All sites")
  2. Dans le menu gauche, clique sur "Site settings" (icône ⚙️)
  3. Dans le sous-menu, clique sur "Domain management"
  4. Descends jusqu'à la section "Options"
  5. Clique sur "Edit site name"
  6. Dans le champ, tape le nom que tu veux (ex: karine-atomy)
    • ✅ Uniquement lettres minuscules, chiffres, tirets
    • ❌ Pas d'espaces, pas d'accents, pas de majuscules
    • ⚠️ Le nom doit être UNIQUE sur tout Netlify
  7. Clique sur "Save"
  8. ✅ Ton nouveau lien : https://karine-atomy.netlify.app
⚠️ Attention :
Après avoir changé le nom, l'ancien lien (random-name-12345.netlify.app) ne fonctionnera plus. Mets à jour tous tes partages !

🔄 Étape 7 : Mettre à jour ton site (après modifications)

  1. Modifie ton fichier index.html sur ton ordinateur
  2. Enregistre les modifications
  3. Va sur Netlify → Clique sur ton site → Onglet "Deploys"
  4. En haut de la page, trouve : "Need to update your site? Drag and drop your site folder here"
  5. Glisse ton dossier COMPLET à nouveau dans cette zone
  6. Attends 30 secondes → ✅ Site mis à jour !
💡 Astuce Pro :
Garde toujours une copie de ton fichier index.html sur ton ordinateur. Comme ça, si tu fais une erreur, tu peux revenir en arrière !

✅ Étape 8 : Tester ton site

  1. Ouvre ton lien Netlify dans un navigateur : https://karine-atomy.netlify.app
  2. Teste avec le paramètre parrain : ?parrain=karine à la fin de l'URL
  3. Vérifie que :
    • ✅ Le nom du parrain s'affiche en haut
    • ✅ Le champ "Qui vous a invité" est pré-rempli
    • ✅ Le formulaire fonctionne
    • ✅ Les liens WhatsApp/email sont corrects
  4. Remplis le formulaire avec ton email de test
  5. Vérifie que tu reçois l'email de confirmation
  6. ✅ Tout fonctionne ? Ton site est prêt à partager !
⚠️ Problèmes fréquents et solutions :
  • "No index.html found" → Ton fichier n'est pas à la racine du dossier ou ne s'appelle pas exactement index.html
  • "Page not found" (404) → Vérifie que le fichier s'appelle bien index.html (minuscules, pas Index.HTML)
  • Le formulaire ne marche pas → Vérifie ta clé API Brevo dans le code JavaScript
  • L'email ne part pas → Vérifie que ton automatisation est ACTIVE dans Brevo
  • Le déploiement est lent → Netlify met 30-60 secondes, sois patient !
