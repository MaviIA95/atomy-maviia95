2. Déploiement sur Netlify

🔄 Mettre à jour ton site

  1. Va sur https://app.netlify.com
  2. Clique sur ton site ATOMY
  3. Va dans l'onglet "Deploys" (menu gauche)
  4. En haut, trouve : "Need to update your site? Drag and drop your site folder here"
  5. Glisse ton dossier atomy-tunnel/ COMPLET (pas les fichiers un par un !)
  6. Attends 30-60 secondes
  7. ✅ Ton site est mis à jour !

🔗 Personnaliser le nom du site

  1. Dans Netlify, clique sur ton site
  2. Menu gauche → "Site settings" (⚙️)
  3. Sous-menu → "Domain management"
  4. Descends à "Options" → "Edit site name"
  5. Tape le nom souhaité : laurent-atomy
  6. Clique sur "Save"
  7. ✅ Nouveau lien : https://laurent-atomy.netlify.app
⚠️ Attention :
• Le nom doit être UNIQUE sur tout Netlify
• Uniquement lettres minuscules, chiffres, tirets (pas d'espaces)
• L'ancien lien ne fonctionnera plus après le changement

4. Gestion des membres

✅ Ajouter / Modifier / Supprimer

Ajouter : Formulaire → "💾 Ajouter" → Générer code → Mettre à jour index.html → Redéployer

Modifier : Clique sur "✏️" → Modifie → "💾 Ajouter" (écrase l'ancien) → Génère → Mets à jour → Redéploie

Supprimer : Clique sur "🗑️" → Confirme → Génère → Mets à jour → Redéploie

🔗 Créer les liens de parrainage

Pour chaque membre, le lien est :

https://laurent-atomy.netlify.app/?parrain=CLE

Exemples :

  • Laurent : ?parrain=laurent
  • Arsène : ?parrain=arsene
  • Alain : ?parrain=alain

5. Configuration Brevo

📊 Voir qui a amené quel lead

  1. Va sur https://app.brevo.com → Contacts → Contacts
  2. Clique sur "Personnaliser les colonnes" (en haut à droite)
  3. Coche la colonne PARRAIN
  4. Clique sur "Appliquer"
  5. ✅ Tu vois maintenant qui a invité chaque contact

📤 Exporter les leads par parrain

  1. Dans la liste des contacts, clique sur "Ajouter un filtre"
  2. Choisis PARRAIN → "est égal à" → "Arsène Ngoran"
  3. Clique sur la coche ☑️ en haut à gauche (sélectionner tous)
  4. Clique sur "Exporter" → Format CSV
  5. Envoie ce fichier au parrain concerné

8. Résolution de problèmes

❌ Le formulaire ne fonctionne pas

  • Ouvre la console (F12 → Console) et regarde les erreurs
  • Vérifie ta clé API Brevo dans index.html
  • Vérifie que listIds: [2] correspond à ta liste
  • Vérifie que templateId: 3 correspond à ton template

❌ L'email automatique ne part pas

  • Vérifie que l'automatisation est ACTIVE dans Brevo
  • Vérifie que le template existe et que l'ID correspond
  • Demander au contact de vérifier ses spams
  • Teste manuellement : Brevo → Contacts → Trouve le contact → 3 points → "Envoyer un email"

❌ Netlify ne déploie pas

  • Vérifie que le fichier s'appelle bien index.html (minuscules)
  • Vérifie qu'il est à la RACINE du dossier (pas dans un sous-dossier)
  • Glisse le DOSSIER complet, pas les fichiers un par un
